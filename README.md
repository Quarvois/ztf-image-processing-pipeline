# ZTF Image Processing Pipeline

This repository contains a Python pipeline developed to process 
Zwicky Transient Facility (ZTF) public data.

## Features
- FITS preprocessing (background, masks, zero-point, seeing)
- Reference image construction
- Image subtraction
- Transient detection (in progress)

## Scientific context
This pipeline was developed as part of a personal research project 
following a Master's thesis in astrophysics.
